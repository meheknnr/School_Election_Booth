# 🗳️ Digital School Election Booth

## Diamond English School

A Digital School Election Booth developed using:

- Python Flask
- SQLite
- HTML5
- CSS3
- Bootstrap 5

## Features

- Student Login
- Admin Login
- Student Council Voting
- One Student One Vote
- Live Results
- Candidate Management
- Student Management
- Election Reports

## Default Admin

Username: admin

Password: admin123

## Run

```bash
pip install -r requirements.txt

python app.py
```

Open

http://127.0.0.1:5000
